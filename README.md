# Crypto Price Alert Pro

Crypto Price Alert Pro is a desktop app built with Python and Tkinter that tracks live cryptocurrency prices and notifies you when your target conditions are met.

It uses the CoinGecko API to fetch real-time data and provides simple strategy insights based on price zones and technical indicators.

---

## 🚀 Features

- 📈 Live crypto price tracking (CoinGecko API)
- 🔔 Custom alerts (price above/below target)
- 🧠 Strategy view (RSI + moving averages + entry zones)
- 🔍 Search and filter coins
- 🪙 Load Top 100 coins dynamically
- ⚡ Smart caching & API rate-limit handling
- 🔊 Alarm notification with test button
- 💱 Supports USD and AUD
